To run this example you would need to do the following:

Frontend is NexusJS and was tested under Firefox, Safari and Chrome
Backend is using .NET MVC 3 and was tested under IIS 7

STEP 1:
In your hosts file point api.nexusjs.com and web.nexusjs.com to 127.0.0.1

STEP 2:
Add 2 websites to your IIS
 1) pointing to Backend.Net directory and using api.nexusjs.com. 
 2) pointing to Frontend.NexusJS directory and using web.nexusjs.com. Make sure index.html is default

STEP 3:
Now you can navigate to 
 Twitter Light TESTS: http://web.nexusjs.com/tests.html
 Twitter Light WEBSITE: http://web.nexusjs.com
